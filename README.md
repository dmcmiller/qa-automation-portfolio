# QA Automation Portfolio
This repository documents a structured 26-week transition into QA Automation Engineering. 
Each folder represents a week of study, practice, and delivered assignments.

## Projects

| Week | Topic | Description |
|------|-------|-------------|
| Week 1 | Python Foundations | CSV parsing, data filtering, file I/O |
| Week 2 | Object-Oriented Python | BugReport class, JSON serialization |
| Week 3 | Introduction to Pytest | 14-test suite for BugReport class |
| Week 4 | Advanced Pytest | Fixtures, conftest, parametrize, custom markers |
| Week 5 | Git & GitHub | Branching, PRs, merge conflicts, commit hygiene |
| Week 6 | Phase 1 Capstone | REST API test suite for JSONPlaceholder — Portfolio Piece #1 |

## Tech Stack
- Python 3.12.4
- pytest
- requests
- pytest-html

## Portfolio Piece #1
`week-06-capstone` — A full pytest suite testing the JSONPlaceholder REST API. 
Includes status code validation, schema validation, data validation, fixtures, 
parametrize, and custom markers.
