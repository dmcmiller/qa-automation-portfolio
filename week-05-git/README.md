# Week 5 - Git

## Overview
Week 5 focused on a more in depth look at git and its commands. Concepts learned include merging, pull request workflow, commit messages and conflict resolution. 

As this focused more on commands and understanding rather than coding, no project was built this week.

## Commands to remember
-`git checkout -b branch-name` — create and switch in one step

-`git branch -d branch-name` — safe local delete

-`git push origin --delete branch-name` — remote delete

-`git reset --hard commit-hash` — discard commits and revert to a prior state

-`git remote prune origin` — clean up stale remote references

-`git commit --amend --reset-author` — rewrite the last commit's author info
